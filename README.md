# Modulaci-n-Digital-ASK-FSK-QPSK-Experimental
Las modulaciones digitales son fundamentales en comunicaciones modernas. Este proyecto implementa y valida experimentalmente tres esquemas de modulación digital (ASK, FSK, QPSK) usando ADALM-Pluto y Python, transmitiendo secuencias binarias, recibiendo señales, y analizando constelaciones y espectros.
